Templates are ordinary workbooks you reuse as a starting point. A template is any workbook whose name ends in `.tmpl.xlsx` — you create them yourself, so a template can be anything from a styled invoice to a budget with formulas already in place.

## To start a workbook from a template

1. Open the Calc index page (the **Calc** entry in the left rail).
2. Click **From template…** (top-right of the page), or use **File → New from template…** while editing another workbook.
3. Pick a template from the list. A new workbook opens with a copy of that template's sheets, ready to edit — your changes don't affect the template.

The list shows every template in your organization's Drive, no matter which folder it lives in. If it's empty, no one has created a template yet.

To start from a blank sheet instead, click **New sheet** — no picker required.

## To save a workbook as a template

1. Open the workbook you want to turn into a template.
2. Choose **File → Export as template…**.
3. Pick the folder to save it in and confirm. The workbook is saved as a new file named `<name>.tmpl.xlsx`, leaving your original untouched.

Your latest edits are captured at the moment you export, so there's no need to wait for a save first. The new template appears in the **From template…** list right away.
