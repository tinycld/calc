Press **⌘/** anywhere in a workbook to open the help search palette and start typing — or choose **Help → Keyboard shortcuts** to land directly here. On iPad, shortcuts work with an external keyboard but not the soft keyboard.

The reference below is the complete set.

## Clipboard

| Shortcut | Action |
|---|---|
| ⌘C | Copy |
| ⌘X | Cut |
| ⌘V | Paste |
| ⌘⌥V | Paste values only |
| ⌘⌥⇧V | Paste formulas only |
| ⌘⇧V | Paste format only |
| ⌘⌥T | Paste transposed |
| Esc | Cancel a pending cut |

## Text formatting

| Shortcut | Action |
|---|---|
| ⌘B | Bold |
| ⌘I | Italic |
| ⌘U | Underline |
| ⌘⇧X | Strikethrough |
| ⌘\\ | Clear formatting |

## Find

| Shortcut | Action |
|---|---|
| ⌘F | Open Find |
| Enter / ▼ | Next match |
| ⇧Enter / ▲ | Previous match |
| Esc | Close Find |

## Undo / redo

| Shortcut | Action |
|---|---|
| ⌘Z | Undo |
| ⌘⇧Z | Redo |

## Editing

| Shortcut | Action |
|---|---|
| Enter | Commit and move down |
| Tab | Commit and move right |
| ⇧Enter | Commit and move up |
| ⇧Tab | Commit and move left |
| F2 | Edit in place |
| Esc | Discard edit |
| Delete | Clear cell content |

## Selection

| Shortcut | Action |
|---|---|
| Arrows | Move active cell |
| ⇧+Arrows | Extend selection |
| ⌘+Arrows | Jump to edge of data |
| ⌘A | Select all |
| ⌘ click | Add to selection (disjoint) |

## Help

| Shortcut | Action |
|---|---|
| ⌘/ | This keyboard shortcuts dialog |
| ? | Help drawer |

## Adding new shortcuts

Calc's shortcuts are gated — they only fire when the selection is in a writable state, the find dialog is closed, etc. If a shortcut seems unresponsive, check that the grid has focus (click a cell) and that no dialog is in front.
