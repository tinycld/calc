## New spreadsheet

**File → New spreadsheet** creates a blank workbook in your Drive and opens it. The new file is named "Untitled spreadsheet" — rename it from the workbook title or from Drive.

## Open

**File → Open** takes you back to the Calc index, where you can pick another workbook. Drive's search and browse work as usual.

## Import

**File → Import** brings outside data in — see [CSV import and export](help://calc:csv-import-export).

## Make a copy

**File → Make a copy** clones the workbook. The current `.xlsx` blob is duplicated into a new Drive item, and the copy opens in a new tab/window so you keep the original handy. The copy is independent — edits don't propagate back to the source.

## Rename

**File → Rename** opens an inline editor on the workbook title at the top of the editor. Type a new name and press **Enter**. The new name applies immediately and propagates to Drive.

You can also click the workbook title directly to rename in place.

## Move to trash

**File → Move to trash** deletes the workbook. It goes to Drive's trash, where you can restore it within the retention window. After that it's permanently deleted.

Deleting a workbook removes everyone's access — collaborators see the room close.

## Details

**File → Details** opens a side panel with:

- Last modified time and who made the change.
- Created time and who created it.
- Owner.
- File size.
- A link to the underlying Drive item (where you can manage sharing, versions, etc.).

## Sharing

Sharing is managed from Drive. **File → Share** jumps you to the Drive share dialog for this workbook. See [Drive sharing](help://drive:sharing).
